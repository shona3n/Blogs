介绍ubuntu的使用…

使用技巧

快捷键

  • 显示隐藏文件：ctrl+h；
  • zsh窗口复制：ctrl+insert；
  • zsh窗口粘贴：shift+insert；

    查看cpu信息

    查看是否支持64位，cpu架构arm或者x86,x86_64等：
  • cat /proc/cpuinfo | grep flags | grep lm | wc -l：0表示支持64位；
  • uname -a: 64位会有x86_64标识；

    查看内存命令

  • free；
  • df: 查看分区内存信息，内存单位KB；
  • df -h：查看分区内存信息，内存单位MB；
  • cat /proc/meminfo: 机器的内存使用信息；
  • du -sh (指定目录)：查看当前目录（指定目录）总大小；
  • ls -ll 或者 ls -lh：查看当前目录文件的大小，前者单位为B，后者单位为KB、MB显示；

查看可执行文件的相关信息

  • file 可执行文件：可查看可执行文件是ARM架构还是X86架构；
  • nm 可执行文件：可查看文件中的符号，包括全局变量，全局函数等；
  • ldd 可执行文件：可查看文件执行所需要的动态库；
  • strings 可执行文件：可查看文件中所有的符号，包括编译器版本信息；
  • readelf 可执行文件或者库文件：可查看文件的所有详细信息，包括文件的头信息，动态库信息，段信息等；
    • readelf -h library.a：查看文件架构等信息；

查看库文件函数接口

nm -g --defined-only *.a，或者*.so；

权限查看与修改

  • 修改文件夹以及子文件夹的读写权限：chmod -R 777 /dir1/dir2；
