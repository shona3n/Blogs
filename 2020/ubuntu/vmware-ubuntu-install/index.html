虚拟机安装ubuntu

首先官网下载ubuntu

打开虚拟机，点击创建新的虚拟机，

选择自定义(高级)，下一步即可，

一直点击下一步，选择 稍后安装操作系统，

保持默认设置，下一步，

设置虚拟机名字，以及保存位置，可使用默认，下一步

后面是内存等的配置，根据电脑需求配置，保持默认也可以。

直到此处，选择网络类型

  • 桥接模式：是会占用宿主计算机所处的网络环境中的IP地址的，虽然他会占用IP地址，但是可以和同网络环境下的设备进行相互通信。

  • NAT模式：该方式不会占用宿主计算机所处网络环境中的IP地址，他会在宿主计算机中虚拟出一个网络环境，给宿主计算机和虚拟机各分配一个IP地址，也就是说这时的宿主计算机是有两个IP地址的，一个是宿主计算机所处的网络环境中的IP地址，一个是和虚拟机构建的虚拟网络环境中的IP地址，这种方式下，虚拟机可以通过宿主计算机的网络代理访问外界资源，但是外界设备不能访问宿主计算机中的虚拟机。

  • 主机模式：这种网络方式，虚拟机只能和宿主计算机进行通信，而不能访问外界设备，更不用说占用IP地址资源了。
    比较常用的是桥接和NAT模式，这里我选择NAT模式.

保持默认设置，以下选择创建新虚拟磁盘，

存储内存分配，不小于默认分配即可，可以根据需要增加，(拆分与不拆分区别：如果以后要移动这个虚拟机，选择拆分。不移动，选择不拆分)
