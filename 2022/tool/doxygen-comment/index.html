doxygen注释规范

发表于2022-08-04|更新于2022-08-04
|阅读量:

Doxygen的工作环境准备好之后，如果想要使用Doxygen就必须在源码的注释中使用Doxygen要求的注释规则，不然Doxygen是不能识别的。

常用注释关键词

文件注释

/**

  • @file demo.h
  • @brief 功能简介
  • @detail 详细描述
  • @author Shona
  • @version 0.1
  • @date 2022-07-29
  • @copyright Copyright (c) 2022
  • @par History:
  • Date Version Author Description
  • 2022-07-29 0.1 Shona
  • /
